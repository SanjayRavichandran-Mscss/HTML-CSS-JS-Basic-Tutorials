FONT FAMILY

syntax:
font-family:times;
font-family:"Times New Roman",Times,serif;
font-family:"Arial",Times,sans-serif; [here,Times is a backup font.The font-family is Times New romanSuppose that font-family not display in a page,The font-family(Times) is display.so that we use the backup font-family(Times)]
font-family:arial,times,sans-serif;

serif:Serif fonts have small lines at the ends on some characters.
Sans-serif:"Sans" means without-These fonts do not have the lines at the ends of the characters.

FONT STYLE

*Normal:The Text is Shown normally.
*italic:The Text is shown in italic.
Syntax:font-style:italic;
The Default Value is normal.

FONT SIZE

Syntax:font-size:50px;

FONT SHORT-HAND PROPERTY

To shorten the code,it is also possible to specify all the individual font properties in one property. Syntax:font:bold 60px arial,sans-serif;
