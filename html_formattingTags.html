*COMMENTS
*BOLD text
*Italic text
*Marked text
*Smaller text
*Deleted text
*Subscript text
*Superscript text.




*Comments SYMBOL------>between the tags Exlamation symbol,double hyphen, any text,double hyphen

*Bold text------> between the tag b

*Italic text------> between the tags i

*Marked text------> between the tags mark

*Smaller text------> between the tags small

*Deleted text------> between the tags del

*Subscript text------> between the tags sub

*superscript text------> between the tags sup
