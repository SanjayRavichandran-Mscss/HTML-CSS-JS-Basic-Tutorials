TYPE-COERCION

Type coercion in JavaScript refers to the automatic conversion of values from one data type to another when performing operations.

let a="25"; //This is string.
let b=30; // This is Number. So, The a & b are concatenate with each other.
console.log(a+b); // output:2530
a=Number(a); // Here,we can convert the String into Number.
console.log(a+b); // So,The Two Numbers are Adding. the output is :55
console.log(typeof a); //output: number
