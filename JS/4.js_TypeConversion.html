TYPE CONVERSION

In Javascript,Type conversion refers to the process of converting the value from one Datatype to another. Javascript is a dynamically Typed Language,which means that variable can hold values of different types.And also the type of value can be changd at Run time.

let a=25;
a=String(a); /* It can be convert from number to string */
console.log(a,typeof a); /*output : 25 string */

This is one of the method to convert the Data type.

let a=50;
console.log(a,typeof a); /* output:50 'number'*/
a=a.toString(); /* This line convert from number to string */
console.log(a,typeof a); /* output: 50 string */

This is Another method to convert the Data type.
