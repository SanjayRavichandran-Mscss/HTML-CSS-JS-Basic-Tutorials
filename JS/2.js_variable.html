variable declaration

var
SCOPE:- It perform both locally and globally.
VARIABLE REDECLARATION:- It allows Redeclaration the values to variable
VALUE ASSIGNMENT:- It allows re-assignment [Assignment means,assign the value to variable after first-declaration,without using keyword(var,let,const)]
let
SCOPE:- It perform only within a block(locally).
VARIABLE REDECLARATION:- It does not allow Redeclaration the values to variable,which means it is constant,not to be changed.
VALUE ASSIGNMENT:- This also allow re-assignment
const
SCOPE:- It also perform only within a block(locally).
VARIABLE REDECLARATION:- It is similar to the 'let' keyword, It is also not allow the Redeclaration the values to variable,which means,it is constant.
VALUE ASSIGNMENT:- 'const' does not allow re-assignment ,But in the object i.e,(key:value) can assign the value.
