Nullish Coalescing Operator

The Nullish coalescing operator (??) is a feature introduced in JavaScript
to provide a concise way to handle nullish (null or undefined) values.
It allows you to provide a default value when accessing a variable or property that may be null or undefined.

var a=null??'NO VALUE';
console.log(a); // OUTPUT : NO VALUE
var b=60??'NO VALUE';
console.log(b); //OUTPUT: 60
const user={'name':'SANJAY','department':'software_system','college':'kgisl' };
console.log(user); //OUTPUT: {name: 'SANJAY', department: 'software_system', college: 'kgisl'}
user.name??='NO_NAME';
user.city??='NO_CITY';
console.log(user.name); //OUTPUT : SANJAY
console.log(user.city); //OUTPUT : NO_CITY
user.department??='Here, We have not mentioned the department'
console.log(user.department); // software_system
